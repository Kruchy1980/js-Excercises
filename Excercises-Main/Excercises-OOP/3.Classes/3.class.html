Class - What is it in JS??


Classes appeared in JavaScript together with ES6 version in 2015th. Class we can say that is a box/container for constructor and all its methodes and properties which lets us keep code more eye/user/programmer ;) friendly. Class is a such an element which lets us use such a behaviours like inheritance, encapsulation or polymorphism
