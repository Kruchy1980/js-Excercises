Classes and Inheritance of child class/subclass prototype.


The parent class/super class is a class from which the child class/sub class inherits properties and methodes. The superclass contains the properties and methodes which can be 'borrowed' to the child classes/subclasses.


Creating of inherit class prototype.


During the inheritance there is a prototype chain appeared - it means that every subclasses has connection to each other by the superclass prototypes.


Calling of constructor of superclass.


The constructor of superclass is called in the subclass/es that is why the subclass/es have the access to every properties and methodes from superclass, moreover we can add any method by creating the prototypes inside the subclasses - remember about line of inheritance (always from parent class).


Extends.


The keyword which shows the parent class from which the subclass will be inherited any elements. Creating object from subclass is equal to calling the constructor of superclass as well.


Super.


The keyword is used for extend the constructor of subclass/es and all other 'higher' subclasses by constructor of superclass.
