Private and Public Properties


All the methodes and properties of objects are public in definition, but we sometimes want and can make them private. It is connected with encapsulation of data.


Hiding Data.


We have two methodes to hide data of our objects. One is Convention and the second one is the usage of the Closures.


Convention.


The mechanism do not really hides the data but informs the programmers that the data are private so do not change them directky inside the object.


Closures usage.


That is the second method of hiding object data by using closures of data in methodes/functions. If the interface is created we can use our property which won't exists when the interface - our class will be erased. Need to remember that methodes used to get or set private datas need to be created inside the Constructor not the Prototype.
