But just to show how czn it look like look below

            1. Create constructor
            const Animal = function(name) {
                this.name = name;
                this.children = [];
            }
            2. Create objects
            const hamster = new Animal('Teofil');
            const canary = new Animal('Singer');
            3. Now we can create methode outside constructor but for it:
            Animal.prototype.addChildren = function(childName) {
                this.children.push(childName);
            }
            4. Now we can create any property outside constructor - rather avoid to do it and implement all properties into constructor:
            Animal.prototype.age = 5; //just as an example

            hamster.addChildren('Hardy');

            //Result for age
            canary.age // 2
            hamster.age // 2
