What is the instance and what constructor is doing.

            //Constructor (pattern, instruction)
            const Animal = funcrion(name, species) {
                this.name = name;
                this.species = species;
            }
            //--------------------------------------
            //Instance Creating
            const dog = new Animal('azor', 'owczarek');
  • Instance is an object created accorcing with the constructor pattern.
  • The emerging object is an independent entity that has its own properties and methods given to it by the reference object (constructor) and has access to the methods and properties owned by such reference object (by referring to the prototype of the constructor).
  • In many object-oriented programming languages we have the structure as the class on which objects are created. In JavaScript, this reference structure (which is a constructor function, but also ES6 is a class) is a set of instructions that are executed on the newly created object.
