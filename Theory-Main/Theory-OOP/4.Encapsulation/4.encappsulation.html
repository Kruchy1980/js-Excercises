Encapsulation

Encpsulation is responsible for:
1. Grouping methodes and properties in object.
2. Controlling of access should be done by obejct itself.
3. Using methodes get and set in objects to control dara flow in it.
4. Hiding implementation by object and giving access for other object with interface - which is public.

Encapsulation - example 

            Math.floor(2.22);

            // We do not care about implementtion and we do not have the access for it. The inrerface for an object math in an object method - floor in that case.

Not hidden data (minimal encapsulation connected with datas without hinding them) - example 

            const user = {
                age: 20 // the value is not protected - can be changed
            }
            // We can change the value of object
            user.age - 21; // value change.
